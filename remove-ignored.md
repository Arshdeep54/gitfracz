# remove-ignored

1. Setup for this exercise

   ```
   git start remove-ignored
   ```  
  
3. Remove the ignored.txt from git using ```git rm ```

   ```
   git rm ignored.text
   ```  

4. Stage and commit the changes 

   ```
   git add ignored.txt
   git commmit -m "remove-ignored"
   ```
