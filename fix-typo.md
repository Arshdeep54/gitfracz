# fix-typo

1. Setup for this exercise

   ```
   git start fix-typo
   ```  
  
3. Edit and stage the file 

   ```
    vim file.txt # edit 
    git add file.txt
   ```  

4. Commit the changes with ``` --amend  ``` to chanhge the commit message

   ```
   git commit --amend -m "Add hello world"
   ```

Now ```git verify```   
Passed!
